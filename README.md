# Sistema de Irrigação Automatizada

> **🎓 Projeto Acadêmico - Disciplina de Microcontroladores**
> Este repositório contém o projeto prático desenvolvido para a disciplina de Microcontroladores (curso de Ciência da Computação). O objetivo principal é aplicar conceitos de sistemas embarcados, integração de hardware/software e análise de dados no mundo real.

## 📌 Sobre o Projeto
Este projeto consiste em um sistema de irrigação inteligente e automatizado, com um forte enfoque na coleta de dados e monitoramento em tempo real. O sistema realiza a leitura constante da umidade do solo, controla a ativação de uma bomba de água utilizando lógica de histerese. O projeto incorpora conceitos avançados de máquinas de estados e processamento multitarefa.

## 🛠️ Tecnologias e Softwares
* **Linguagem:** C/C++ (Framework Arduino / ESP-IDF)
* **Sistema Operacional de Tempo Real:** FreeRTOS (gerenciamento de multitarefas)
* **Dashboard Web:** HTML, CSS e JavaScript
* **Conectividade:** WebSockets para comunicação em tempo real

## 🔌 Componentes de Hardware
* **Microcontrolador:** ESP32
* **Sensor:** Sensor de Umidade de Solo Higrômetro
* **Atuador:** Mini bomba De Água – Diafragma – 12v RS385
* **Controle de Potência:** Módulo Relé
